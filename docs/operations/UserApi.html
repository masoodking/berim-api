This is the API

userActivatePost

after signup we should send an activation code to user by sms and user send it back to us by this function

URL

https://api.berim.com/v1/user/activate

HTTP Method

POST

Response Type

List

Parameters

  • Query: phoneNumber BigDecimal

    user phone number.

  • Query: activationCode BigDecimal

    activation code.

userGetProfileGet

The User Profile endpoint returns information about the user that has authorized with the application.

URL

https://api.berim.com/v1/user/get_profile

HTTP Method

GET

Response Type

User

Parameters

  • Query: phoneNumber BigDecimal

    user phone number.

  • Query: password String

    user password.

userResendActivationCodePost

each user need to activate his/her account so we send an activation code to user that user should enter it in client after signup

URL

https://api.berim.com/v1/user/resend_activation_code

HTTP Method

POST

Response Type

List

Parameters

  • Query: phoneNumber BigDecimal

    user phone number.

userSetProfileGet

همه داده‌های ارسالی ذخیره و جایگزین داده های قبلی می‌شوندو در صورتی که داده‌ای ارسال نشده یا خالی ارسال شده است در نظر گرفته نمی‌شود.

URL

https://api.berim.com/v1/user/set_profile

HTTP Method

GET

Response Type

User

Parameters

  • Query: full_name String

    full name of the user.

  • Query: picture String

    Image avatar URL of the user.

  • Query: username String

    username of user.

  • Query: phoneNumber BigDecimal

    user phone number.

  • Query: password String

    user password.
